## 2. تدفقات أساسية

- [ ] تسجيل الدخول بحساب مدير
- [ ] تسجيل الدخول بحساب كاشير
- [ ] فتح وردية جديدة
- [ ] إنشاء طلب جديد من شاشة `POS`
- [ ] إنهاء طلب نقدي
- [ ] إنهاء طلب بطاقة
- [ ] طباعة فاتورة العميل
- [ ] طباعة تذكرة المطبخ
- [ ] استقبال طلب أونلاين من الموقع
- [ ] وصول تنبيه الطلب للكاشير
- [ ] عرض الطلب في شاشة `KDS`
- [ ] تنفيذ دفع أونلاين ناجح
- [ ] التحقق من تحديث حالة الدفع

## 3. المشتريات والمخزون

- [ ] إنشاء أمر شراء
- [ ] إنشاء سند استلام مورد

- [ ] مراجعة سجل حركة المخزون
- [ ] التأكد من ظهور المورد والمستودع بشكل صحيح

## 4. المحاسبة

- [ ] التأكد من إنشاء القيد الآلي بعد عملية البيع
- [ ] التأكد من إنشاء القيد الآلي بعد استلام المشتريات
- [ ] مراجعة دفتر الأستاذ للحسابات المتأثرة
- [ ] مراجعة رصيد الصندوق أو البنك بعد الدفع
